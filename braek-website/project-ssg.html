Identidade Visual & Web

Overview

O SSG Saúde é um ecossistema completo de saúde digital, focado em unificar a linguagem visual entre médicos e pacientes. O desafio era criar uma interface que transmitisse confiança, agilidade e modernidade, mantendo a acessibilidade para todos os usuários.

Desenvolvemos um Design System robusto com mais de 200 componentes, garantindo escalabilidade para o produto e consistência visual em todas as telas — do agendamento online ao painel administrativo.
